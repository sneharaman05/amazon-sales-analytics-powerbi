📊 Amazon Global Sales Analytics Dashboard | Power BI

🧾 Project Overview

Amazon Global Sales Analytics Dashboard built in Power BI to analyze 38,000+ sales records, track key business KPIs, forecast sales trends, and identify high-performing markets using advanced data modeling techniques.

This project focuses on turning raw sales data into meaningful insights for better business decision-making.

🎯 Key Objectives
Analyze global sales performance across regions and categories
Track important KPIs like revenue, profit, and return rate
Identify top-performing markets and products
Build sales forecasts to support business planning
Improve reporting efficiency using automation in Power BI

🛠️ Tools & Technologies Used
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Excel (Data Source)
Data Modeling

📊 Key Features of Dashboard
📌 Interactive Sales Overview Dashboard
📈 KPI Cards (Revenue, Profit, Orders, Returns)
🌍 Regional Performance Analysis
📦 Product Category Insights
🔁 Return & Profitability Analysis
📉 Sales Trend Forecasting
🎯 Dynamic Filters (Region, Category, Time Period)

🔍 Key Insights
Identified top-performing regions contributing highest revenue
Found seasonal sales trends affecting business performance
Analyzed return patterns impacting profitability
Highlighted best-selling product categories
Discovered growth opportunities in underperforming markets
West region generated highest sales
Electronics category dominates revenue
Loss observed in low-margin products
